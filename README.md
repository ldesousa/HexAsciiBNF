Description
--------------------------------------------------------------------------------------

Defines a grammar to encode hexagonal grids in ASCII text files.

Please consult the BNF file for the exact meaning of each parameter.

Example
-------

```
nclos	3
nrows	3
xll		2.5
yll		-2.5
side	1.5
no_data	9999
23 24 23.2
24 24.2 23.7
24.1 23.8 23.3
```
